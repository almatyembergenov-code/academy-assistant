# B2C Test Control — MVP v0.2

Визуально обновленный черновой прототип для обработки ежемесячного тестирования фронт-офиса B2C.

## Что умеет первая версия

- Загружать файл “Не начат” из iSpring.
- Готовить очищенный файл для тренеров.
- Объединять несколько файлов от тренеров с кодами причин.
- Загружать итоговую выгрузку результатов.
- Загружать историю последних тестов.
- Делать простой расчет:
  - неуважительная причина → 0%;
  - уважительная причина + 3 оценки в истории → средний балл;
  - уважительная причина без истории → 80%;
  - спорные случаи → ручная проверка.
- Формировать базовый свод и выгружать итоговый Excel.

## Как запустить

Открой `index.html` в браузере.

Важно: для работы чтения/записи Excel используется SheetJS с CDN:
https://cdn.jsdelivr.net/npm/xlsx@0.18.5/dist/xlsx.full.min.js

Для корпоративной/офлайн-версии библиотеку лучше скачать локально и подключить из папки проекта.
